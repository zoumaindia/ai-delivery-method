# Validation Brief

**Mode:** Read-only  
**Goal:** Validate the system and produce an execution-ready task list.

## Required output

1. Confirmed bugs
2. Root-cause mapping
3. Enhancement opportunities
4. Questions only if truly needed
5. Execution-ready task list

## Rules

- Do not implement fixes in this pass.
- Prefer reproducible findings over speculative ones.
- Separate known issues from newly discovered issues.
- Support browser findings with backend evidence where applicable.
