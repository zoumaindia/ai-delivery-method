# Project Starter

Copy this starter into a new software project and adapt it for your team.

## Starter contents

- startup docs
- task and decision structure
- smoke-check scaffold
- thin AI wrapper examples

## Minimal startup flow

1. Fill `docs/overview.md`
2. Fill `docs/conventions.md`
3. Fill `docs/working-agreement.md`
4. Create the first `DECISIONS.md` entries
5. Add the first workstreams to `docs/TASKS.md`
6. Use a task brief before major feature work
